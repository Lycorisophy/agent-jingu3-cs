# v0.1 部署文档

## 1. 环境要求

- JDK 17+
- Maven 3.8+（或使用 IDE 内建 Maven）
- 可选：本地 [Ollama](https://ollama.com/)，并拉取默认模型（见 [jingu3-开发规范.md](../规范/jingu3-开发规范.md)）

## 2. 配置

- 主配置：[agent-jingu3-cs-server/src/main/resources/application.yml](../../agent-jingu3-cs-server/src/main/resources/application.yml)
- 关键项：
  - `jingu3.ollama.base-url`：默认 `http://localhost:11434`
  - `jingu3.ollama.chat-model`：主模型名
  - `jingu3.user.id` / `jingu3.user.name`：单用户种子

## 3. 本地运行

```bash
cd agent-jingu3-cs-server
mvn spring-boot:run
```

默认 `spring.profiles.active=dev`（见 `application.yml`）。

## 4. 构建

```bash
cd agent-jingu3-cs-server
mvn -q -DskipTests package
java -jar target/agent-jingu3-cs-server-1.0-SNAPSHOT.jar
```

## 5. 依赖服务

- **Ollama**：未启动时，意图分类与 Ask/ReAct 调用将失败或降级（分类降级为 REACT，对话仍可能报错）；开发环境请先启动 Ollama 并确认模型已拉取。

## 6. 回滚

- 回退到上一 Git 标签/提交；重新部署对应 `jar`；无数据库迁移（v0.1 无持久化）。
