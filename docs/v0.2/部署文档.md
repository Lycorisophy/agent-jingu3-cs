# v0.2 部署文档

## 1. 环境要求

- JDK 17+
- Maven 3.8+（或 IDE 内建 Maven）
- 可选：本地 [Ollama](https://ollama.com/) 与默认模型（见项目规范）

## 2. 配置

- 主配置：[application.yml](../../agent-jingu3-cs-server/src/main/resources/application.yml)
- v0.2 新增：
  - **`jingu3.cron.demo-schedule`**：Cron MVP 演示用调度表达式字符串（默认 `0 0 9 * * MON-FRI`），仅用于回复文案，**不触发真实调度**。

其余与 v0.1 相同：`jingu3.ollama.*`、`jingu3.user.*`。

## 3. 本地运行

```bash
cd agent-jingu3-cs-server
mvn spring-boot:run
```

## 4. 构建

```bash
cd agent-jingu3-cs-server
mvn -q -DskipTests package
java -jar target/agent-jingu3-cs-server-1.0-SNAPSHOT.jar
```

## 5. 依赖服务

- **Ollama**：Plan-and-Execute、Workflow、Agent Team、Ask、ReAct、意图分类等依赖 LLM；未启动时调用可能失败。
- **无数据库**（v0.2）；状态追踪为进程内 Map，重启清零。

## 6. 回滚

- 回退 Git 提交；替换 `jar`；无 DB 迁移。
