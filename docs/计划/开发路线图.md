## v0.1 — 引擎内核与路由契约（史诗① 启动）

**版本状态**：已完成（2026-04-12）

**目标**：Spring Boot 工程可运行；**意图场景识别**三源路由与优先级；**ModeRegistry** 与八模式统一契约；Ask/ReAct 等至少部分真跑通；最小会话上下文（**不计入**记忆系统史诗完成）；用户常量 `001`/`user`。

### 文档与工序（`docs/v0.1/`）

- [x] PRD → 可行性 → 概要 → 详细 → 服务端代码 → 接口文档 → 前端（本版本跳过）→ 单测 → 部署文档

### 功能需求

- [x] `SYS:意图路由` 三源路由接口：客户端显式 > 规则 > 模型
- [x] `SYS:行动模式` ModeRegistry + 八模式接口；至少 Ask、ReAct 可演示
- [x] `SYS:行动模式` 其余模式 Stub 或占位返回（接口齐全）
- [x] `ENG:提示词` 基础提示词模板与拼装
- [x] `ENG:上下文` 引擎所需最小上下文（非记忆史诗）
- [x] `ENG:驾驭` 路由决策日志/可观测
- [x] Spring Boot：替换占位 Main、健康检查、配置分层（如 dev/prod）
- [x] 单用户 `001` / `user` 常量或配置

**本版本关闭条件**：上列全部勾选；`docs/v0.1/` 文档工序齐全。
