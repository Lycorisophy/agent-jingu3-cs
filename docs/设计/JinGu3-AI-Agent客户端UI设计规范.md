## 一、概述

### 1.1 产品定位

JinGu3 AI Agent 是一款基于 C/S 架构的智能助手客户端，支持多种 AI 行动模式，提供记忆管理、技能工具、全局定时任务等功能。

### 1.2 设计目标

- **简洁高效**：最小化操作步骤，快速完成任务
- **功能强大**：支持 6 种行动模式，满足复杂任务需求
- **可扩展**：插件化技能系统，支持自定义开发
- **智能提醒**：卡片消息 + 桌面通知，不遗漏重要信息

### 1.3 名词定义

| 术语 | 定义 |
|------|------|
| JinGu3 | 产品名称 |
| Agent | AI 智能体 |
| 行动模式 | Agent 处理任务的策略方式 |
| 技能 | Agent 可调用的工具能力 |
| 记忆 | Agent 的上下文存储体系 |
| 定时任务 | 自动化执行的计划任务 |
