# 指南 §1.4 / §3 与路线图对齐说明

本文件说明《AI智能体行动模式设计指南》中 **记忆管理（§1.4）**、**Ask 与工具（§3）** 与本项目 [开发路线图](../计划/开发路线图.md) 的对应关系；**不在本文件实现代码**，仅作版本认领与验收锚点。

**说明（路线图 v2.0）**：差距整改将 **工具子系统 MVP** 前移至 **v0.3**；**记忆系统**顺延至 **v0.6**；**技能系统扩展**为 **v0.7**。下文表格已按新编号更新。

## §1.4 记忆分层（STM / Episodic / LTM）

| 指南小节 | 路线图落点 | 说明 |
|----------|------------|------|
| 短期记忆、截断 | **v0.6 记忆系统** | `ExecutionContext` 与检索注入策略在详细设计中展开 |
| 情节/状态轨迹 | v0.6 + **v0.5** State 持久化 | State 横切在 v0.5 先行最小表，记忆史诗补齐检索语义 |
| 向量/RAG/LTM | **v0.6**（Qdrant + 嵌入） | 对齐指南 §1.4.4 |

## §3 Ask 与工具

| 指南小节 | 路线图落点 | 说明 |
|----------|------------|------|
| 工具选择 / Router | **v0.3 工具 MVP** | Tool Registry、执行管线、Ask/ReAct 真闭环 |
| 工具 Schema / 企业级技能 | **v0.7 技能扩展** | 在 v0.3 基础上扩展 JSON Schema、客户端工具、安全分级 |
| 多轮上下文 + 工具 | v0.6 记忆 + v0.3/v0.7 工具 | `AskModeHandler` 从「纯 LLM」演进为「路由→工具→汇总」，并由记忆增强上下文 |
