# 指南 §1.4 / §3 与路线图对齐说明

本文件说明《AI智能体行动模式设计指南》中 **记忆管理（§1.4）**、**Ask 与工具（§3）** 与本项目 [开发路线图](../计划/开发路线图.md) 的对应关系；**不在本文件实现代码**，仅作版本认领与验收锚点。

## §1.4 记忆分层（STM / Episodic / LTM）

| 指南小节 | 路线图落点 | 说明 |
|----------|------------|------|
| 短期记忆、截断 | **v0.3 记忆系统** | `ExecutionContext` 与检索注入策略在详细设计中展开 |
| 情节/状态轨迹 | v0.3 + State 辅助能力 | 与 `STATE_TRACKING` 引擎侧协同 |
| 向量/RAG/LTM | v0.3（Qdrant + 嵌入） | 对齐指南 §1.4.4 |

## §3 Ask 与工具

| 指南小节 | 路线图落点 | 说明 |
|----------|------------|------|
| 工具选择 / Router | **v0.4 技能工具系统** | Tool Registry、Schema、执行管线 |
