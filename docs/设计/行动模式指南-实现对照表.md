## 附录：意图路由行为（对应指南「选型」入口）

1. 若请求含非空 `modePlan` → 顺序执行（见 `ModePlanExecutor`），不跑三源路由。
2. 否则：客户端显式 `mode` → `IntentRouter`；无效字符串 → `FALLBACK`+`REACT`；不可对话模式 → `400`。
3. 无显式 mode：关键词规则 → `RuleBasedModeRouter`；否则 LLM 分类 → `ModelIntentClassifier`（仅允许对话可选五模式）。
