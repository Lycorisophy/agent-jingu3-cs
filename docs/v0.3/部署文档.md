# v0.3 部署说明

在 [v0.1](../v0.1/) / [v0.2](../v0.2/) 部署方式基础上，v0.3 增加工具子系统与可选生产日志卷。

## 1. 运行前置

- **JDK 17+**
- **Ollama**：默认 `http://localhost:11434`，模型见 `application.yml` 的 `jingu3.ollama.chat-model`
- **Maven**：构建与门禁执行 `mvn clean verify`（含 PMD、SpotBugs）

## 2. 配置要点

| 配置 | 说明 |
|------|------|
| `jingu3.tool.enabled` | 生产若需与旧行为完全一致可置 `false` |
| `jingu3.engine.react.max-steps` | ReAct 最大步数，默认 4 |
| `spring.profiles.active` | 置为 `prod` 时启用 `log4j2-spring.xml` 中 **RollingFile**（`logs/jingu3.log`，按日/大小滚动） |

## 3. 构建与启动

```bash
cd agent-jingu3-cs-server
mvn clean package -DskipTests
java -jar target/agent-jingu3-cs-server-1.0-SNAPSHOT.jar
```

开发profile 默认 `dev`，仅控制台日志；生产建议使用 `prod` 并挂载 `logs/` 目录。

## 4. 健康检查

`GET /actuator/health`（与 v0.1 相同）。
