# v0.4 概要设计

## 1. 模块

| 模块 | 变更 |
|------|------|
| `engine.support.ToolStepService` | Plan 子任务与工作流 TOOL 节点的路由/执行/汇总复用 |
| `PlanAndExecuteModeHandler` | 子任务执行链接 `ToolStepService`；流式发 `TOOL_RESULT` |
| `WorkflowModeHandler` / `workflow` 包 | `WorkflowNode.type` / `toolId`；LLM/TOOL 分支 |
| `prompt` | Plan 子任务路由与工具后汇总模板 |
| `tool` | 无破坏性变更，复用 v0.3 |

## 2. 兼容性

- 无 `type` 的 workflow 节点 → LLM。
- `jingu3.tool.enabled=false` 时 Plan/Workflow 行为与 v0.3 前一致（子任务纯 LLM）。
