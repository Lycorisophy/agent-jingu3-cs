# v0.4 部署说明

在 [v0.3 部署文档](../v0.3/部署文档.md) 基础上，v0.4 **不新增中间件**。

## 1. 构建

```bash
cd agent-jingu3-cs-server
mvn clean verify
```

## 2. 工作流定义文件

- 路径：`src/main/resources/workflows/*.json`（打包后仍在 classpath）。
- 新增示例：`demo-wf-tool.json`（含 `TOOL` 节点）；联调时请求 `workflowId=demo-wf-tool` 且 `mode=WORKFLOW`。

## 3. 配置

- `jingu3.tool.enabled`：与 v0.3 相同；关闭后 Plan/Workflow 工具路径退化。
- `jingu3.engine.plan.max-subtasks`、`jingu3.engine.plan.replan-enabled`：与 v0.2 相同。
