# v0.5 概要设计

## 1. 模块划分

| 模块 | 职责 |
|------|------|
| `cron` / `schedule` | 任务持久化、触发、执行委托 |
| `hitl` | 队列、REST、与 Chat 衔接 |
| `state` | 任务状态 CRUD |
| `engine/mode` | `CronModeHandler` 升级或侧路 API；`HumanInLoopModeHandler` 与队列联动；`StateTrackingModeHandler` 持久化；`AgentTeamModeHandler` 可选增强 |

## 2. 部署

- 单进程 Spring Boot 即可；Quartz 可选。
